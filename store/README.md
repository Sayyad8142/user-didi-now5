# Didi Now — Store Submission Pack

Use these files for App Store Connect and Google Play Console. Edit any text before submission.

**Important:** Both stores require a **Privacy Policy URL** on the web. In-app PDFs are great, but you still need a public URL for the listing (e.g., a simple page on your website or Notion/GitHub Pages). Keep the text consistent with `/legal` inside the app.

- For iOS metadata: see `store/ios/*`
- For Android metadata: see `store/play/*`
- Icons/Splash: see `resources/README.md`
