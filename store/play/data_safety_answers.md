**Data collected**
- Personal info: name, phone
- App activity: bookings (service type, time, status)
- Optional files: user never uploads; admins may upload worker photos

**Data sharing**
- Not sold; shared only with assigned worker as needed to fulfill a booking

**Security**
- Encrypted in transit; row-level security for data access

**Deletion**
- Users can delete account and data from Profile → Account & Data

**Optional permissions**
- None required for users (tel/UPI links open system apps)
